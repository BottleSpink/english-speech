Welcome!


This is a demo application to practice English speech. Only Chrome browser is supported.


  • Select text what you want to read aloud and click "Continue".
  • On next screen push "Record" and hold to record your voice.
  • The voice will be recognized via Web Speech API and you will see how perfectly you speak.

Keep in mind
Recognition does not always work well.
Some phrases can be recognized incorrectly even with perfect pronunciation.


If you have any questions, comments or suggestions, just let me know lb.im@yandex.ru or visit project page.
